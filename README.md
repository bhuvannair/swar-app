# Swar — Pitch & Swara Tuner

Live web app: **https://bhuvannair.github.io/swar-app/**

A browser pitch/swara tuner with dual Western + Indian notation, audio-file note-sequence
analysis, raga detection, MIDI/CSV export, and an ear trainer.

This repository holds only the built, self-contained `index.html` for hosting on GitHub
Pages. The source lives in a separate (private) repository and is bundled into this single
file via its build step.
